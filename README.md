# TNTP -> GR

To convert `tntp` to `gr` files use `tntp2gr.py` script:

```
python tntp2gr.py source.tntp values.gr cost.gr
```

`tntp2gr` split `tntp` into 2 `gr` files where first contains values, second - cost
